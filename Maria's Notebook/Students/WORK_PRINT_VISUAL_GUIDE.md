## Layout Features

### Header Section
- **Report Title**: Bold, prominent at the top
- **Date**: Current date when printed
- **Filter Description**: Shows what items are included
- **Sort Description**: Shows how items are ordered
- **Summary Stats**: Total items and student count

### Student Sections
- **Student Name**: Bold section header
- **Work Items**: Numbered list under each student
- **Dividers**: Clear separation between students

### Work Item Details
Each work item shows:
1. **Index Number**: Sequential numbering for easy reference
2. **Subject Indicator**: Color dot (● in grayscale print)
3. **Lesson Name**: The lesson this work is for
4. **Work Type & Status**: Practice/Follow Up/Report, Active/Review
5. **Due Date**: With overdue indicators in red
6. **Checkbox**: Empty box (☐) for manual tracking
7. **Title**: Custom work item title if present
8. **Notes**: Brief notes if present (truncated to 2 lines)
9. **Step Progress**: For report-type work (e.g., "✓ 2/4 steps completed")

### Footer
- Simple attribution line

## Design Goals

### Paper Efficiency
- **Compact layout** without sacrificing readability
- **Small but legible fonts** (11-12pt for body, 10pt for secondary)
- **Minimal whitespace** between items
- **Strategic use of spacing** only where it aids comprehension
- **No unnecessary graphics** or decorations

### Readability
- **Clear hierarchy** through font weights and sizes
- **Logical grouping** by student
- **Visual separators** using divider lines
- **Color indicators** (print as grayscale if B&W printer)
- **Consistent formatting** across all items

### Practicality
- **Checkboxes** for physical tracking
- **Due date prominence** for priority planning
- **Student grouping** for differentiated instruction
- **Notes inclusion** for context
- **Overdue highlighting** for immediate action items

## Print Settings Recommendations

### For Best Results
- **Orientation**: Portrait
- **Paper**: US Letter (8.5" × 11")
- **Margins**: 0.5" on all sides (handled automatically)
- **Color**: Color prints best, but grayscale works well
- **Quality**: Standard quality sufficient

### Paper Conservation
- Approximately **8-12 work items fit on one page**
- For larger lists, consider:
  - Filtering to urgent items only
  - Printing front/back if printer supports duplex
  - Reducing margins slightly in print dialog

## Use Cases

### Daily Planning
Print your "Due Today" filter to have a physical reference during lessons.

### Weekly Review
Print all open work to review with students or plan upcoming lessons.

### Parent Conferences
Print work for specific students to discuss progress and plans.

### Physical Records
Some educators prefer physical backups of digital work tracking.

### Offline Work
Take printed lists to locations without device access.

## Customization Notes

If you need to adjust the layout, edit `WorkPrintView.swift`:

- **Font sizes**: Search for `.system(size:` calls
- **Spacing**: Adjust `.padding()` and `spacing:` parameters
- **What's shown**: Comment out sections you don't need
- **Grouping**: Change the `groupedWork` computed property
- **Page size**: Modify the `612 x 792` dimensions

The layout is designed to be easily modifiable while maintaining the core goal of paper efficiency and readability.
